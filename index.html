MEMORY

Qrystals can store a unique memory directly on the blockchain

SCANNABLE

Use a smartphone camera to scan a qrystal to access its memory

EDITABLE

A qrystal owner can modify its memory an unlimited number of times

PURITY

Qrystals can have varying purities which affect their rarity

UNIQUE

No two qrystals are alike, even if they have the same purity

READ MEMORY

Point your smartphone camera to a qrystal and follow the popup, it should take you to its memory.
If the memory is a link, you will be redirected to the link, otherwise you will see its memory in the penseive.
